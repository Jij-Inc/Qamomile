core.converters.converter#

qamomile/core/converter.py

This module defines the QuantumConverter abstract base class for converting between different problem representations and quantum models in Qamomile.

The QuantumConverter class provides a framework for encoding classical optimization problems into quantum representations (e.g., Ising models) and decoding quantum computation results back into classical problem solutions.

Key Features: - Conversion between classical optimization problems and Ising models. - Abstract methods for generating cost Hamiltonians and decoding results. - Integration with QuantumSDKTranspiler for SDK-specific result handling.

Usage: Developers implementing specific quantum conversion strategies should subclass QuantumConverter and implement the abstract methods. The class is designed to work with jijmodeling for problem representation and various quantum SDKs through the QuantumSDKTranspiler interface.
