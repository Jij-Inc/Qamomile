# Qamomile v0.11.1

Qamomile v0.11.1は、サポートするPythonの下限を**3.11**まで下げたリリースです。ライブラリの挙動に変更はなく、Python 3.12 / 3.13でv0.11.0を使っているユーザーへの影響はありません。

```
pip install qamomile==0.11.1
```

## ハイライト

- **Python 3.11サポート。**`requires-python`を`>=3.12`から`>=3.11`に緩和し、CIでも`3.11`、`3.12`、`3.13`をUbuntu / macOS両方で（CUDA-Qマトリクスを含めて）テストするようになりました（#335）。

## 補足

- `requires-python`に上限は設定されていないため、Python 3.14でもインストール自体は可能で、動作すると見込まれます。ただし現時点ではCIマトリクスに含めていないため、公式な互換性保証はしていません。3.14がCIに追加されるまでは自己責任でご利用ください。
